# 融資

---

## 基本的な考え方

**8割近く融資で買う。これが不動産投資の最大の強み。**

- 手元のキャッシュをほとんど減らさずに資産を増やせる
- 融資でレバレッジを効かせながら、減価償却で法人税を節税できる唯一の投資手法
- → [節税](03_節税) 参照

---

## 「先に融資を引いておく」の重要性

**お宝物件は突然やってくる。即動ける状態を作っておかないと取れない。**

**実例（那珂市：エルグランドナカ）**
> コロナの頃、融資がガバガバだった時期に2,000万円ほど融資を先に引いていた。そこへ楽街経由で紹介メールが届き、師匠に速攻で連絡して見学→OKが出たので即買付。一番手での買付だったから取れた。

- **2,000万円台ならいつでも買える状態**にしておいた
  - 自治金融：1,000万円
  - コロナ融資：1,000万円

### なぜ事前融資が有効か

- お宝物件は好条件ゆえに競争が激しい
- 「融資が下りるかどうか確認してから」では遅い
- 現金買付に近い速度で動くためには、事前に枠を作っておくことが必須

---

## 融資の種類と活用実績

| 種類 | 金額 | 特徴 |
|------|------|------|
| 自治金融（制度融資） | 1,000万円 | 地方自治体経由の低利融資 |
| コロナ融資 | 1,000万円 | コロナ期間中の特例。ガバガバだった |
| 銀行融資（通常） | 物件ごとに変動 | 金利・期間は交渉次第 |

---

## 融資条件の考え方

### 金利

- **10年固定金利より20年変動金利の方がCFが貯まりやすい**（AIシミュレーション結果）
- 金利が低いほど毎月返済額が減り、手残りのCFが増える

### 返済期間

- 長期（20年）にするほど毎月の返済が小さくなり、CFが大きくなる
- ただし総返済額は増える

### 実際の計算例（石岡市の物件）

| 項目 | 数値 |
|------|------|
| 購入価格 | 2,680万円（表面利回り18.8%） |
| 融資 | 2,010万円（10年返済・金利1.5%） |
| 自己資金 | 約670万円 |
| 毎月返済 | 約18万円 |
| 毎月CF | 約8.7万円 |
| 年間CF | 104万円（実質利回り15%） |
| ローン完済後のCF | 約30万円/月（約3倍） |

---

## 30年シミュレーションの視点

- 家賃下落を考慮しても**累計CFは5,000〜6,000万円規模**
- ローン完済後はCFが一気に拡大する
- **買う前の段階でCF・税効果ともにほぼ読める「確定利回り型投資」**に近い

---

## 融資を使うメリットまとめ

1. **少ない自己資金で大きな資産を取得できる**
2. **返済が進むほど純資産が増加する**（元本返済＝強制的な貯蓄）
3. **減価償却と組み合わせると法人税をほぼゼロにできる**
4. **キャッシュフローが毎月手元に残る**
