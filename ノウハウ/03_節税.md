# 節税

---

## 基本的な考え方

**不動産投資は収益と節税が同時に効く唯一の投資手法。**

- 減価償却費を計上することで法人税を大幅に圧縮
- 消費税を比例按分方式で実質ゼロ化できる
- 8割融資で買っているので手元キャッシュがほとんど減らない
- → 「税金で消えるはずだった本業の利益を、頭金として再投資し、資産とキャッシュを増やすサイクルに変換できる」

---

## スキーム全体像：築古×建物按分×短期償却

### ステップ1：利回りの高い耐用年数超え（または近い）物件を購入

- CFが出やすい
- 減価償却の償却期間を短くできる（= 毎年の経費を大きく取れる）

### ステップ2：不動産鑑定士に「意見書」を依頼

- **建物割合を高めるための正式な根拠として活用**
- 税務調査が入っても問題にならないところまで限界まで攻める
- 依頼タイミング：申告の2ヶ月前でOK
- 複数物件まとめて依頼すると割安になる
- 実績：**3物件で合計44万円**

### ステップ3：建物割合を高く設定する

通常（固定資産税評価額基準）：
```
土地7：建物3
```

鑑定士の意見書を活用すると：
```
土地3：建物7（逆転できる）
```

→ 経費化できる減価償却費が大幅に増える

---

## 法定耐用年数と償却期間

| 構造 | 法定耐用年数 | 耐用年数超えの場合の償却年数 |
|------|------------|--------------------------|
| 木造 | 22年 | 残存年数×20%（最短4年） |
| 軽量鉄骨 | 19〜27年 | 同上 |
| 重量鉄骨 | 34年 | 残存年数×20% |
| 鉄筋コンクリート | 47年 | 同上 |

→ 耐用年数超え物件は、**最短4年で全額償却できる場合もある**

## 消費税のほぼゼロ化

### 仕組み：比例按分方式

- 通常、居住用不動産は非課税 → 仕入れ時の消費税控除ができない
- **「個別対応方式」ではなく「比例方式」を選ぶと、本業の課税売上との割合で控除できる**
- ただし建物価格**1,000万円未満**が条件（1,000万円以上は控除不可）

### 2026年9月30日まで：インボイス特例を活用する

- 2026年9月30日まで、**インボイス未登録事業者との取引でも80%控除可能**
- この期間に不動産を仕入れると消費税を大幅圧縮できる

**計算例（建物価格980万円、課税売上割合90%）**
```
980万円 × 90% × 80% = 約705,600円の控除
```

### 狙い目の物件スペック

- 購入価格：1,500万円前後
- 築古
- 土地3：建物7（建物比率が高い）
- 利回り15%以上

## 節税 × 出張旅費規定の活用

法人であれば出張旅費規定を活用することで、さらに節税できる。

| ケース | 日当 | 特徴 |
|--------|------|------|
| 日帰り（4時間以上の外出） | 10,000円 | 所得税・社会保険不要。近所のカフェでもOK |
| 宿泊 | 20,000円 ＋ 交通費（グリーン車可）＋ 宿泊費（都内3万円） | — |

**月10日・日帰り外出 ＋ 月1回都内宿泊の場合：年間200万円近く節税可能**

### 不動産×旅費規定の合わせ技

- 物件を見に行くという名目で**家族旅行も経費にできる**

---

## まとめ：本業 ＋ 不動産賃貸業の組み合わせが最強な理由

1. 少ない資金をレバレッジで大きな資産に変えられる
2. キャッシュフローが毎月安定して入る
3. 減価償却で法人税を大幅に圧縮できる
4. 消費税もほぼゼロ化できる
5. **税金で消えるはずのお金を頭金に再投資するサイクルが作れる**
6. **広告で利益を生み → 不動産に変える循環が最強コンボ**
